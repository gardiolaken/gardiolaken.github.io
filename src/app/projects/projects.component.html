HomeWork Covid Help
JavaScript,HTML,CSS
I made this project at the start of covid. I realized that my nieces were getting too "Ipady", watching youtube and playing games. So I decided to make this app that their parents can use.
  1. How it works is the parent is to create a "quiz".
  2. To create a quiz, they need to provide a "pass" (I will explain the use of this later).
  3. Select Difficulty.
  4. After creating the quiz, a SessionKey will be generated.
  5. The parents are to provide this key to their kid/s.
  6. The kid is then to use this SessionKey to take the quiz.
  7. Once the kid successfully finishes the quiz with a good result.
  8. The "pass" that the parent provided during the creation will be shown to them.
  9. The kid can then tell the pass to their parents to prove that they've passed the quiz.
I think this can be improve to automatically have a security app on their devices. Involving some home network engineering.
